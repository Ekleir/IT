##  INSERT INTO SELECT
Данный оператор копирует данные из одной таблицы и вставляет их в другую, при этом типы данных в обеих таблицах должны соответствовать.
```clike
INSERT INTO second_table
SELECT * FROM first_table
WHERE condition;
```
